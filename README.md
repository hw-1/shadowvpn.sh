# shadowvpn.sh
### shadowVPN 一键安装脚本
shadowVPN 一键安装脚本 支持最新 Ubuntu 24.04.2 LTS   内存消耗低， 可使用最低配制的VPS

``` bash
curl -o  installer.sh "https://raw.githubusercontent.com/hw-1/shadowvpn.sh/refs/heads/main/installer.sh" &&\
chmod a+x installer.sh &&\
sudo   ./installer.sh
```
